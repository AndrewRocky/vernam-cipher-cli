# vernam-cipher-cli
Vernam cipher implementation in python3
## Usage:
```
python.exe .\vernam_gamma.py -h
usage: vernam_gamma.py [-h] -i INPUT -k KEY -o OUTPUT (-e | -d)

Powerless tool for encryption/decryption in Vernam's cypher    

optional arguments:
  -h, --help            show this help message and exit
  -i INPUT, --input INPUT
                        Input file
  -k KEY, --key KEY     Private key input file
  -o OUTPUT, --output OUTPUT
                        Output file
  -e, --encrypt         encryption mode
  -d, --decrypt         decryption mode
```
